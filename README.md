# Arena Master: League & Knockout Showdown

Arena Master is a Tournament Management System developed using C, C++, Flask, and Data Structures.

## Features
- Team Registration
- Round Robin Fixture Generation
- Knockout Tournament Management
- Leaderboard Generation
- Match Result Recording
- File Handling for Data Storage
- GUI Interface using Qt
- Web Interface using Flask

## Data Structures Used
- Linked Lists
- Trees
- Queues
- Stacks
- Graphs
- Sorting Algorithms

## Technologies Used
- C
- C++
- Python (Flask)
- Qt Framework
- File Handling

## Project Modules
- Backend Tournament Engine
- GUI Interface
- Web Interface
- Knockout Bracket Generator
- Leaderboard System
